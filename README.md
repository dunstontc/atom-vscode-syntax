# dark-plus-syntax theme

Visual Studio Code's `Dark+` (default dark) theme, ported to Atom. <br>

## Supported Languages:

- [x] HTML
- [x] CSS
- [x] LESS
- [x] SASS & SCSS
- [ ] Stylus
- [x] Markdown
- [ ] GFL
- [x] JavaScript
- [ ] JavaScript (Babel)
- [ ] Javascript React (Babel) (JSX)
- [x] JSON & CSON
- [x] CoffeeScript
- [ ] TypeScript
- [ ] VUE
