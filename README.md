# dark-plus-syntax theme

Visual Studio Code's `Dark+` (default dark) theme, ported to Atom. <br>

## Supported Languages:

  - [x] HTML
  - [ ] CSS
  - [ ] LESS
  - [ ] SASS & SCSS
  - [ ] Markdown
  - [ ] GFL
  - [ ] JavaScript
  - [ ] Babel / JSX
  - [x] JSON & CSON
  - [ ] CoffeeScript
  - [ ] TypeScript
  - [ ] VUE
