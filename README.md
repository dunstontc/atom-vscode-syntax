# dark-plus-syntax theme

Visual Studio Code's `Dark+` (default dark) theme, ported to Atom. <br>  

## Supported Languages:

  - [ ] HTML
  - [ ] CSS
  - [ ] LESS
  - [ ] SASS & SCSS
  - [ ] Markdown
  - [ ] GFL
  - [ ] JavaScript
  - [ ] CoffeeScript
  - [x] JSON & CSON
  - [ ] TypeScript
  - [ ] JSX
  - [ ] VUE
